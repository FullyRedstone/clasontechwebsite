About Us

Our Goal

Our goal is to create websites and applications in a reasonable amount of time for reasonable prices with great customer satisfaction.

What we do

We create websites and applications at a affordable price and deliver top of the line customer service. We understand the struggles of time.

What do we use?

We are willing to use any website development software such as wix, squarespace or wordpress, although we prefer to just code it in html and css.

Pricing

Wix, Squarespace, Wordpress, ect

For easy site creation software the pricing is pretty cheap. The standard cost is 80$. Plus any more for any other features. Note when using these it costs more per month to host the website than coding it would.

Coding it in html and css

Since this takes a lot more time and effort than using a site creator, the pricing is going to be a lot higher. The standard price is 200$ for just html and css. If you would like java-script and a interactive experience, this can take much longer and is more complex. For those projects there is an additional 200$.

Hosting the website using our web servers.

Have we coded your site in html or you have an html version of your site but nowhere to host it? Well come no farther. We have an initial charge of 10$ a month for hosting the site, plus 1$ per 100 visitors a day. Also depending on how big the site is (file size), we may charge more per month due to limited storage capacity.

Minecraft Servers

We are even willing to create a minecraft server for you. Depending on the type of server prices will vary. The initial price will be 200$ since it takes a lot of time to create the server. We are also willing to host the server for negotiable specs and prices.

Software and application development

Since software and application development takes a lot of time and a lot of effort, the pricing for application development is a lot more. The initial charge is 400$, plus 100$ per two weeks on the application development.
